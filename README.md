# Goal

## Parser

### Input

* CupFiles
    * cup.txt
    * cup_finals.txt

#### cup.txt structure
* Cup name begins with =
* 3 \n
* Groups with its composition, separated by |
* 3 \n
* Matchdays
* 4 \n
* Groups of matchs 
    * Match composition
